# To-Do List Project

This is a simple To-Do list application built using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, and remove tasks from the list. It's a lightweight and easy-to-use web app that demonstrates basic front-end development skills.

## Features

- **Add Tasks**: Users can add tasks to the to-do list.
- **Mark as Completed**: Users can mark tasks as completed.
- **Remove Tasks**: Users can remove tasks from the list when they are no longer needed.
- **Persistent Storage**: The tasks are stored in the browser's local storage so that they persist even after refreshing the page.

## Technologies Used

- **HTML**: Structure of the application.
- **CSS**: Styling of the application.
- **JavaScript**: Logic for adding, removing, and updating tasks, as well as local storage functionality.
