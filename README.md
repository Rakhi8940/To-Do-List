**To-Do List Web App**
This is a simple, interactive To-Do List web application where users can add, mark as completed, and delete tasks. The app features a sleek design with hover effects, smooth transitions, and responsive layout adjustments for various screen sizes.

**Features**
1. Add Tasks: Users can type and add tasks to their to-do list.
2. Mark as Completed: Click on a task to mark it as completed (crosses out the text).
3. Delete Tasks: Users can delete tasks from the list.
4. Responsive Design: The app is optimized for mobile, tablet, and desktop screen sizes.
5. Interactive Elements: Buttons have hover effects, and tasks animate when deleted.

**Technologies Used**
1. HTML: Structure and layout of the To-Do list.
2. CSS: Styling the application, including the responsive design and hover effects.
3. JavaScript: Handling the dynamic functionality for adding, completing, and deleting tasks.

**Usage**
1. Add a Task: Type your task in the input field and click the Add button (or press Enter).
2. Complete a Task: Click on a task to mark it as completed. This will strike-through the text and change the background color.
3. Delete a Task: Click the trash bin icon next to a task to delete it from the list.
